# 3WAYS
</br>

## 프로그램 설명
Unity를 이용해서 만든 2D게임. 
기본 조작은 가상 조이스틱을 이용해서 총알 피하기와 같지만 특정 시간마다 상하좌우 네 방향중 하나의 방향으로 갈 수 없게된다. 
중간에 나오는 너트를 이용해서 체력을 회복 시킬수도 있다.
</br>
## 사용언어

C#, Unity
